# PRUEBA TÉCNICA - CRUD PHP/JavaScript

## Requisitos Previos
- PHP 8.2 o superior
- MySQL Workbench 8.0 o superior instalado 
- MySQL Server 9.0.1 o superior
- Navegador web moderno

## Instalación

1. **Configuración de la Base de Datos**:
   - Ejecutar el script SQL ubicado en /sql/schema.sql
   - Este creará la estructura de la base de datos y la tabla necesaria

2. **Configuración del Entorno**:
   - Editar `config/database.php` con tus credenciales de base de datos

## Ejecución de la Aplicación

1. **Iniciar Proyecto WEB PHP**:
   - Abrir la terminal/consola en la carpeta raíz del proyecto
   - Ejecutar el comando: php -S localhost:8000

2. **Acceder a la Aplicación**:
   - Abrir navegador web
   - Ingresar a la dirección: http://localhost:8000
